# Inventari i Dyqanit
Projekt praktik për lëndën **Programim Dot-Net**.
Një aplikacion Console në C# që menaxhon produktet e një dyqani me operacionet bazë CRUD. Të dhënat ruhen në një klasë statike (pa databazë).
## Operacionet
- **Create** – shton një produkt
- **Read** – shfaq produktet
- **Update** – ndryshon një produkt
- **Delete** – fshin një produkt
## Si ekzekutohet
Duhet të kesh të instaluar .NET SDK. Pastaj, te dosja e projektit:
```
dotnet run
```
