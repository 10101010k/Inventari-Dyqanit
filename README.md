# Inventari i Dyqanit

Projekt praktik për lenden **Programim Dot-Net**.

Nje aplikacion Console ne C# që menaxhon produktet e nje dyqani me operacionet baze CRUD. Te dhenat ruhen ne nje klase statike (pa databaze).

## Operacionet

- **Create** – shton nje produkt
- **Read** – shfaq produktet
- **Update** – ndryshon nje produkt
- **Delete** – fshin nje produkt

## Si ekzekutohet

Duhet te kesh te instaluar .NET SDK. Pastaj, te dosja e projektit:

```
dotnet run
```
